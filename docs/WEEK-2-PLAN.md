## ▎Day 6-7: 实现子任务

### 功能需求
1. 任务详情页添加"添加子任务"按钮
2. 子任务列表组件
3. 父任务显示完成进度（3/5 已完成）
4. 完成所有子任务后，父任务自动完成
5. 支持多层嵌套（最多 3 层）

### 数据模型
```typescript
// tasks 表已有 parent_id 字段
interface Task {
  id: string;
  title: string;
  parentId: string | null; // 父任务 ID
  // ... 其他字段
}
```

### 实施步骤
1. 创建 SubTaskList.tsx 组件
2. 创建 TaskDetail.tsx 组件
3. 添加子任务 CRUD 操作
4. 计算父任务完成进度
5. 自动完成逻辑

### 验收标准
- ✅ 可以添加子任务
- ✅ 子任务可以独立完成
- ✅ 父任务显示完成进度
- ✅ 支持多层嵌套（最多 3 层）
- ✅ 完成所有子任务后，父任务自动完成
