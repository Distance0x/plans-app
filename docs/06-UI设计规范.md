# UI 设计规范

## 设计理念

**核心风格**：Modern Glassmorphism + Minimalist

**设计原则**：
1. **简洁至上**：去除一切不必要的装饰
2. **层次分明**：通过毛玻璃效果营造空间感
3. **流畅动画**：所有交互都有平滑过渡
4. **一致性**：统一的间距、圆角、配色系统

## 毛玻璃效果（Glassmorphism）

### 标准卡片
```css
.glass-card {
  background: rgba(255, 255, 255, 0.7);
  backdrop-filter: blur(12px);
  border: 1px solid rgba(255, 255, 255, 0.3);
  border-radius: 16px;
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.1);
}

/* 深色主题 */
.dark .glass-card {
  background: rgba(31, 41, 55, 0.7);
  border: 1px solid rgba(255, 255, 255, 0.1);
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.3);
}
```

### 悬浮效果
```css
.glass-card:hover {
  background: rgba(255, 255, 255, 0.8);
  box-shadow: 0 12px 48px rgba(0, 0, 0, 0.15);
  transform: translateY(-2px);
  transition: all 0.3s ease-out;
}
```

## 响应式设计

### 断点
```css
--breakpoint-sm: 640px;
--breakpoint-md: 768px;
--breakpoint-lg: 1024px;
--breakpoint-xl: 1280px;
```

### 布局
- **桌面**（>= 1024px）：侧边栏 + 主内容区
- **平板**（768px - 1023px）：可折叠侧边栏
- **手机**（< 768px）：底部导航栏

## 无障碍设计

### 对比度
- 正常文本：至少 4.5:1
- 大文本（18px+）：至少 3:1
- 交互元素：至少 3:1

### 焦点状态
```css
.focus-visible {
  outline: 2px solid var(--primary-500);
  outline-offset: 2px;
}
```

### 键盘导航
- Tab：焦点移动
- Enter/Space：激活按钮
- Esc：关闭模态框
- Arrow Keys：列表导航

## 吉祥物设计

### 形象
- **名称**：Tomo（番茄的昵称）
- **造型**：可爱的番茄形象
- **尺寸**：128x128px（标准）

### 状态动画
1. **专注状态**：眼睛专注，轻微呼吸动画
2. **休息状态**：眼睛闭上，放松姿态
3. **庆祝状态**：跳跃动画，星星特效
4. **空闲状态**：眨眼，轻微摇摆

### 实现方式
使用 **Lottie 动画**（JSON 格式）

```tsx
import Lottie from 'lottie-react';
import tomoFocus from '@/assets/animations/tomo-focus.json';

<Lottie 
  animationData={tomoFocus} 
  loop 
  className="w-32 h-32"
/>
```
