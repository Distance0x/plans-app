# 项目概述

## 项目名称
Plans App - Windows 桌面任务管理应用

## 项目简介
基于 Electron 架构开发的 Windows 桌面任务管理应用，类似滴答清单，集成番茄钟、日历视图、每日清单和 AI 虚拟形象功能，提供高效的时间管理和任务追踪体验。

## 核心功能

### 1. 番茄钟（Pomodoro Timer）
- **标准番茄工作法**
  - 25 分钟工作时段
  - 5 分钟短休息
  - 4 个番茄钟后 15-30 分钟长休息
- **自定义时长**
  - 用户可自定义工作时长
  - 用户可自定义休息时长
- **通知提醒**
  - Windows 系统通知
  - 声音提醒（可选）
  - 桌面弹窗提醒
- **统计功能**
  - 今日完成的番茄钟数
  - 本周完成的番茄钟数
  - 历史统计图表
- **后台运行**
  - 最小化到系统托盘
  - 后台计时不中断

### 2. 日历视图
- **多视图切换**
  - 月视图：查看整月任务分布
  - 周视图：查看本周任务详情
  - 日视图：查看当日任务时间线
- **任务拖拽**
  - 拖拽任务调整日期
  - 拖拽任务调整时间
- **日期导航**
  - 快速跳转到指定日期
  - 今日高亮显示
  - 重要日期标记
- **任务展示**
  - 显示任务截止日期
  - 显示任务优先级
  - 显示任务完成状态

### 3. 每日清单
- **任务管理**
  - 创建任务
  - 编辑任务
  - 删除任务
  - 完成状态切换
- **任务属性**
  - 任务标题
  - 任务描述
  - 优先级（高/中/低）
  - 截止日期
  - 标签分类
  - 子任务支持
- **任务操作**
  - 搜索任务
  - 过滤任务（按优先级、状态、标签）
  - 排序任务（按优先级、截止日期、创建时间）
- **任务视图**
  - 列表视图
  - 看板视图（可选）

### 4. 吉祥物动画
- **虚拟角色展示**
  - 可爱的番茄形象（Tomo）
  - Lottie 动画实现
- **表情变化**
  - 专注状态：认真表情
  - 休息状态：放松表情
  - 完成任务：庆祝表情
  - 空闲状态：默认表情
- **交互反馈**
  - 番茄钟结束提醒
  - 任务完成庆祝
  - 鼓励性动画

## 技术架构

### 前端技术栈
- **框架**: Electron 28+
- **UI 框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式**: Tailwind CSS
- **状态管理**: Zustand
- **UI 组件**: Shadcn UI（基于 Radix UI）
- **动画**: Framer Motion
- **日历**: React Big Calendar
- **图标**: Lucide React
- **虚拟滚动**: react-window

### 后端技术栈
- **数据库**: SQLite3
- **ORM**: Drizzle ORM
- **Node.js 绑定**: better-sqlite3

### 吉祥物动画技术
- **动画方案**: Lottie（JSON 动画）
- **动画库**: lottie-react
- **备选方案**: SVG + CSS 动画

### 工具链
- **打包**: electron-builder
- **自动更新**: electron-updater
- **系统通知**: node-notifier
- **日期处理**: date-fns

## 非功能需求

### 性能要求
- 界面响应时间 < 100ms
- 应用启动时间 < 3 秒
- 内存占用 < 200MB（空闲状态）

### 数据持久化
- 本地 SQLite 数据库
- 数据自动保存
- 数据备份与恢复

### 用户体验
- 离线优先，无需网络
- 系统托盘支持
- 开机自启动（可选）
- 主题切换（浅色/深色）
- 快捷键支持

### 安全性
- 本地数据加密（可选）
- 数据导出功能
- 数据导入功能

## 目标用户
- 需要时间管理的知识工作者
- 使用番茄工作法的用户
- 需要任务管理的学生和职场人士
- 喜欢可爱虚拟形象的用户

## 竞品分析
- **滴答清单**: 功能全面，但缺少番茄钟和虚拟形象
- **Forest**: 有趣的番茄钟，但任务管理功能较弱
- **Notion**: 强大的笔记和任务管理，但学习曲线陡峭
- **Microsoft To Do**: 简洁易用，但功能相对基础

## 项目优势
- 集成番茄钟和任务管理
- 可爱吉祥物增加趣味性
- 本地优先，数据安全
- 开源免费
- 可定制性强
- 现代化 UI 设计（Glassmorphism）
