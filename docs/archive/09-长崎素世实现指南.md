## 快速开始 - 使用简化版本

当前应用已经集成了简化版本的吉祥物，可以立即使用：

```bash

npm run dev
```

你会在右下角看到一个圆形的吉祥物，显示"素世"文字，并根据应用状态自动切换动画。

## 升级到完整版本 - 使用真实图片

### 步骤 1：下载长崎素世图片

**推荐来源：**

1. **BanG Dream! Wiki**
   - 访问：https://bandori.fandom.com/wiki/Nagasaki_Soyo

   - 右键保存图片

2. **Bandori Party**
   - 访问：https://bandori.party/
   - 搜索 "Nagasaki Soyo" 或 "長崎素世"
   - 下载官方美术图

3. **Steam Workshop**
   - 访问：https://steamcommunity.com/sharedfiles/filedetails/?id=3429680793
   - 可能有社区制作的资源

### 步骤 2：准备图片资源

你需要准备 4 张图片，对应 4 种状态：

| 状态 | 文件名 | 描述 | 建议表情 |
|------|--------|------|----------|
| idle | soyo-idle.png | 空闲状态 | 微笑、默认表情 |
| focus | soyo-focus.png | 专注状态 | 认真、严肃表情 |
| rest | soyo-rest.png | 休息状态 | 放松、开心表情 |
| celebrate | soyo-celebrate.png | 庆祝状态 | 兴奋、庆祝表情 |

**图片规格：**
- 格式：PNG（透明背景）
- 尺寸：512x512 或 1024x1024
- 文件大小：< 200KB/张
- 背景：透明

### 步骤 3：处理图片

如果下载的图片有背景，使用以下工具去除：

**在线工具：**
- [Remove.bg](https://www.remove.bg/) - 自动去除背景
- [Photopea](https://www.photopea.com/) - 在线 Photoshop

**步骤：**
1. 上传图片到 Remove.bg
2. 自动去除背景
3. 下载透明 PNG
4. 使用 [TinyPNG](https://tinypng.com/) 压缩文件

### 步骤 4：放置图片文件

创建目录并放置图片：

```bash
cd plans-app
mkdir -p src/assets/mascot
```

将 4 张图片放入 `src/assets/mascot/` 目录：
```
src/assets/mascot/
├── soyo-idle.png
├── soyo-focus.png
├── soyo-rest.png
└── soyo-celebrate.png
```
